# FlowStreak SPEC

## Problem
Daily three-wins board that animates streak momentum and suggests the next micro goal.

## Target user
Productivity/inbox apps embedding daily recap surfaces.

## User stories
- As a user, I want to list wins + pick energy/focus.
- As a user, I want to flowstreak renders celebration + next focus cards.
- As a user, I want to copy/share or regenerate.
