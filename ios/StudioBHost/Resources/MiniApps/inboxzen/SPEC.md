# InboxZen SPEC

## Problem
Inbox triage coach summarizing top emails and drafting reply prompts to clear the queue fast.

## Target user
Email/CRM platforms embedding smart triage.

## User stories
- As a user, I want to paste subjects, pick tone/goal/time.
- As a user, I want to review triage list + prompts.
- As a user, I want to copy recap or regenerate.
