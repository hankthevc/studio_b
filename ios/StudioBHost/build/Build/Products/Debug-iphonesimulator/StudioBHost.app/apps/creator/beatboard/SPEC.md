# BeatBoard SPEC

## Problem
Produces short audio hooks/loops tuned to vibe sliders for reels or podcast bumpers.

## Target user
Creator/music tools needing deterministic prompts.

## User stories
- As a user, I want to pick vibe/length/use case/energy.
- As a user, I want to review sonic palette, hook structure, and tips.
- As a user, I want to share or regenerate.

- [ ] Deterministic logic hooked into shared plan builder
- [ ] Shared app shell layout + branded tokens
- [ ] README, EMBED, UX notes, and apple metadata updated
