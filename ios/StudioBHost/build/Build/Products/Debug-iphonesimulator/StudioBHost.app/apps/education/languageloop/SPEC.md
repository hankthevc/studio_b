# LanguageLoop SPEC

## Problem
Phrase-practice loops with roleplay prompts, pronunciation tips, and cultural context for quick language reps.

## Target user
Language-learning apps needing quick practice modules.

## User stories
- As a user, I want to pick language + scenario + confidence.
- As a user, I want to read warm-up, roleplay, and accent cards.
- As a user, I want to copy or regenerate for another scenario.
