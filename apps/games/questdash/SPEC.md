# QuestDash SPEC

## Problem
Generates micro adventure challenges for friends or couples with photo-proof prompts and shareable results.

## Target user
Social/party apps embedding mini challenges.

## User stories
- As a user, I want to choose crew, energy, setting, and proof.
- As a user, I want to questdash outputs kickoff, challenges, and wildcard cards.
- As a user, I want to copy/share or regenerate for another round.

- [ ] Deterministic logic hooked into shared plan builder
- [ ] Shared app shell layout + branded tokens
- [ ] README, EMBED, UX notes, and apple metadata updated
