# TipTally SPEC

## Problem
Tip + split calculator infused with local norms, quick QR payment shortcuts, and round-up suggestions.

## Target user
Finance/payment apps embedding tip calculators.

## User stories
- As a user, I want to enter bill, service vibe, split count, and round-up preference.
- As a user, I want to review tip math, split breakdown, and script cards.
- As a user, I want to copy the share note or regenerate.

## MVP checklist
- [ ] Deterministic logic hooked into shared plan builder
- [ ] Shared app shell layout + branded tokens
- [ ] README, EMBED, UX notes, and apple metadata updated
