# StorySpark UX Notes

## Flow
1. **Hero + form:** Eyebrow `AI content studio`, title `StorySpark`, tagline about hooks/captions. Form: idea textarea, platform select, tone tags (Playful/Bold/Warm), optional CTA input.
2. **CTA interaction:** `Spark My Hooks` validates idea, shows sparkle loader.
3. **Results:** Highlight card (Top hook), alternate hooks list with copy buttons, caption card, hashtag chips, share row, upsell banner.
4. **Regenerate:** Button rebuilds using last inputs/tone.
5. **Error states:** Missing idea shows helper `Drop an idea or headline to get sparks.`
