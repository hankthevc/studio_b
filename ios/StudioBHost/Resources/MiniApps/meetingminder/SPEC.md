# MeetingMinder SPEC

## Problem
Auto-builds prep checklists and post-meeting follow-ups using calendar context so nothing slips.

## Target user
Calendar/project tools embedding meeting companions.

## User stories
- As a user, I want to pick meeting type/goal/duration and list attendees.
- As a user, I want to review three cards: prep, agenda, follow-up.
- As a user, I want to copy plan for the invite or regenerate.

## MVP checklist
- [ ] Deterministic logic hooked into shared plan builder
- [ ] Shared app shell layout + branded tokens
- [ ] README, EMBED, UX notes, and apple metadata updated
