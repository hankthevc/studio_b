# PitchPulse SPEC

## Problem
Elevator-pitch builder that scores clarity, suggests trims, and preps you for lightning intros.

## Target user
Career/networking products embedding pitch prep.

## User stories
- As a user, I want to select audience/tone/time/focus.
- As a user, I want to review draft, trim tips, and follow-up card.
- As a user, I want to copy or regenerate.
