# PulsePath SPEC

## Problem
Adaptive HIIT/interval builder that responds to current energy, available gear, and target burn with guided pacing.

## Target user
Fitness/connected equipment apps embedding quick HIIT brains.

## User stories
- As a user, I want to select session goal, gear, and time block.
- As a user, I want to get a three-block plan with work/rest cues.
- As a user, I want to copy/share or regenerate with new intensity.

## MVP checklist
- [ ] Deterministic logic hooked into shared plan builder
- [ ] Shared app shell layout + branded tokens
- [ ] README, EMBED, UX notes, and apple metadata updated
