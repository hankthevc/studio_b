# SliceSaver SPEC

## Problem
Bill-splitting optimizer for roommates or trip crews with fairness suggestions and gentle reminder scripts.

## Target user
Finance/banking apps embedding smart split flows.

## User stories
- As a user, I want to select crew size and total.
- As a user, I want to choose tone + due to generate plan.
- As a user, I want to copy share note or regenerate.

## MVP checklist
- [ ] Deterministic logic hooked into shared plan builder
- [ ] Shared app shell layout + branded tokens
- [ ] README, EMBED, UX notes, and apple metadata updated
