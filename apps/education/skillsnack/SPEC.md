# SkillSnack SPEC

## Problem
Five-minute micro-lessons covering practical skills (Excel, negotiation, design) you can swipe through daily.

## Target user
Learning platforms or communities needing micro-content.

## User stories
- As a user, I want to choose skill/time/format/confidence.
- As a user, I want to review primer, steps, and apply cards.
- As a user, I want to copy/share or regenerate.

## MVP checklist
- [ ] Deterministic logic hooked into shared plan builder
- [ ] Shared app shell layout + branded tokens
- [ ] README, EMBED, UX notes, and apple metadata updated
