# TempoBuddy SPEC

## Problem
Micro run/ride pacer that designs interval cues, surfaces playlist tempos, and delivers friendly post-run recaps.

## Target user
Running/fitness platforms needing a quick pacing assistant.

## User stories
- As a user, I want to enter terrain, distance, goal vibe, and playlist bpm.
- As a user, I want to tempobuddy outputs warmup, tempo wave, and cooldown cards.
- As a user, I want to share link with training partners or regenerate.

## MVP checklist
- [ ] Deterministic logic hooked into shared plan builder
- [ ] Shared app shell layout + branded tokens
- [ ] README, EMBED, UX notes, and apple metadata updated
