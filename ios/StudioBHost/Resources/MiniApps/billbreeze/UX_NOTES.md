# BillBreeze UX Notes

## Hero & Hook
- Eyebrow: Cash-flow lane
- Title + tagline mirror README copy.

## Form
- Fields: Income cycle, Bills (one per line amount + name), Stress level, Primary goal
- Helper text: Need pay cycle + at least one bill.

## Flow summary
- Select income cadence and list bills.
- Review lane overview, focus bills, and next actions.
- Share link or regenerate when bills change.

## States
- Placeholder: Enter a couple bills to sketch the lane.
- Loading: Shared shell loader.
- Error: Shared toast copy.

## Copy highlights
- Upsell banner: BillBreeze Pro / Unlimited bills, projected balances, and PDF export.
- Share row label: Share cash-flow lane
