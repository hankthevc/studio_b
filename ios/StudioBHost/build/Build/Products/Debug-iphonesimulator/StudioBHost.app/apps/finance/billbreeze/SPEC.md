# BillBreeze SPEC

## Problem
Turns recurring bills into a visual cash-flow lane with upcoming alerts and “safe to spend” guidance.

## Target user
Neobank/budgeting apps embedding cash-flow clarity.

## User stories
- As a user, I want to select income cadence and list bills.
- As a user, I want to review lane overview, focus bills, and next actions.
- As a user, I want to share link or regenerate when bills change.

## MVP checklist
- [ ] Deterministic logic hooked into shared plan builder
- [ ] Shared app shell layout + branded tokens
- [ ] README, EMBED, UX notes, and apple metadata updated
