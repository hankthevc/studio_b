# SipSync UX Notes

## Flow
1. **Hero + form:** Eyebrow `AI hydration coach`, title `SipSync`, tagline about keeping pace. Form: wake time, sleep time, activity select, flavor mood tags (Crisp/Fruity/Herbal), travel toggle, optional notes.
2. **CTA:** `Sync My Sips` validates wake/sleep, shows droplet loader.
3. **Results:** Highlight card with daily target, timeline card listing reminders (time + ounces), flavor idea chips, travel tip banner (if toggle), share row, upsell banner.
4. **Regenerate:** Button rebuilds plan using last inputs.
5. **Error states:** Missing wake/sleep shows helper `Add wake and sleep times to sync hydration.` Clipboard fallback matches TripSpark.
