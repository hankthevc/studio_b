# FlexDesk SPEC

## Problem
Desk-friendly stretch and mobility bursts triggered by time of day or posture prompts to keep remote workers limber.

## Target user
Employee experience teams embedding mobility nudges into their apps.

## User stories
- As a user, I want to enter schedule anchor + tension area.
- As a user, I want to pick energy + duration to see a three-card sequence.
- As a user, I want to copy/share or regenerate with a new vibe.

- [ ] Deterministic logic hooked into shared plan builder
- [ ] Shared app shell layout + branded tokens
- [ ] README, EMBED, UX notes, and apple metadata updated
