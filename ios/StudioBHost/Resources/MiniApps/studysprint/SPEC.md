# StudySprint SPEC

## Problem
Turns study topics into 25-minute sprints with recall prompts, confidence check, and quick recap notes.

## Target user
Study/productivity apps embedding pomodoro-style helpers.

## User stories
- As a user, I want to enter topic, deadline, confidence, and preferred format.
- As a user, I want to review focus plan, recall prompts, and break card.
- As a user, I want to copy/share or regenerate for another topic.

## MVP checklist
- [ ] Deterministic logic hooked into shared plan builder
- [ ] Shared app shell layout + branded tokens
- [ ] README, EMBED, UX notes, and apple metadata updated
