# ClipChoreo SPEC

## Problem
Transforms an idea into a three-shot storyboard with on-screen text guidance and beat suggestions.

## Target user
Creator tools needing polished storyboard scaffolds.

## User stories
- As a user, I want to enter concept/platform/pace/tone.
- As a user, I want to review shot cards and tips.
- As a user, I want to copy share link.
